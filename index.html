What is HTML Semantic Elements?

HTML Semantic elements help building the code we create, making it is easier for the audience to browse while maintaining a good format for developers. Semantic elements help us think about the structure the layout, dynamic of data, and to choose titles' hierarchy properly. They help us differentiate the semantic elements of our markup from the ones we use solely for layout.
